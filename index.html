HTML

Elements

Tag Name Description
<h1> - <h6> Heading Elements - ordered from the highest level to the lowest
<li> List Item Element - element to create a list
<ol> Ordered List - items in sequential order (numbered)
<ul> Unordered List - items in no particular order (bullet point)
<div> Division Element - divides a document into sections
id="" ID Attribute - can be applied to only one element in a page
class="" Class Attribute - can be applied to multiple elements
<p> Paragraph Element - displays a block of text
<span> Span Element - used to group text for styling purposes
<strong> Strong Element - will render text in bold
<em> Emphasis Element - will render text itelicized
<video> Video Element - embeds media player for video playback
<audio> Audio Element - implements audio into a website
<img> Image Element - embeds images in documents
<embed> Embedding Element - can be used to implement any type of media
alt="" Alt Attribute - adds alternative text to <img>
<br> Line Break Element - creates a line break in text
<figure> Figure Element - used to encapsulate media
<figcaption> Figcaption Element - used describe the media within the <figure>
<section> Section Element - defines elements in a document with the same theme
title="" Title Attribute - the text will appear when a user hovers their cursor over the element
<article> Article Element - independent content for a specific theme
<aside> Aside Element - used to mark additional info

Structure

Tag Name Description
<!DOCTYPE html> HTML Declaration - required as the first line for HTML5 doc
<a> Anchor Element - used to create hyperlinks
<head> Head Element - general info about HTML page that isn't displayed
target="" Target Attribute - with value of _blank hyperlink will open in a new tab
href="#" Section link - href with # points to desired id
<html> HTML Element - all HTML content goes in it
<!--comment--> Comments - used to make notes
<title> Title Element - title displayed in browser's tab
<style> Style Element - lets CSS code be written in HTML file
style="" Style Attribute - used for inline styles
<link> Link Element - link HTML docs to external resources
href="" Hypertext Reference - path to specify URL. Local: ./path
rel="" Relation Attribute - used to specify relation. Ex: rel="stylesheet"
type="" Type Attribute - to define the type of content. Ex: type="text/css"

Tables

Tag Name Description
<table> Table Element - all table content goes in it
<tr> Table Row - used to add rows
<td> Table Data - nested inside <tr> to add a cell of data
<thead> Table Head - defines the headings of table columns
<th> Table Heading - used to add titles to rows and columns
<tbody> Table Body - contains all table data other than table heading
<tfoot> Table Footer - uses table rows to give footer content
rowspan="" Rowspan Attribute - indicates how many rows cell should span
colspan="" Colspan Attribute - indicates how many columns cell should span

CSS

Selectors

Selector Name Description
p {} Type Selector - selects all <p> tags
.classname {} Class Selector - selects all elements with class="classname"
#idname {} ID Selector - selects elements with id="idname"
Chaining Description
element1,element2 - all element1 and all element2
element1 element2 - all element2 inside element1
.classname1.classname2 - all elements with classname1 and classname2 in its class=""
.classname .classname2 - all elements with classname2 that is a child of an element with name1

Visual Rules

Pseudo-Class Description
:hover - changes when holding mouse over the item
:focus - matches when an element has focus
:active - matches when an item is being activates by the user
Property Description
!important - used to override other declarations and ignore selector specificity
font-size: - is used to set text size
background-color: - controls background color of elements
font-weight: - used to set the boldness of text
text-align: - used to set the text alignment of inline contents
font-family: - specifies the typeface in a rule set
text-decoration: - adds decorative lines to texts
cursor: - changes mouse cursor
color: - sets the foreground text color of an element
opacity: - ranges from 0 to 1 and controls the transparency of an element
url("") - links resources to properties (can set background-image:)

The Box Model

Property Description
padding: - clears an area around the content
border:
width style color;		 - goes around the padding and content
margin: - clears an area outside the border
margin: auto; - can be used to horizontally center elements
overflow: - determines how browser behaves when content is too large
max-width: - caps size even when width: property is larger
visibility: - renders hidden objects invisible to users without removing them
box-sizing: - sets how total width and height of an element is calculated
border-radius: - used to modify border shape

Display

Property Description
display: block; - element displayed as a block item (like <p>)
display: inline; - element displayed as an inline element (like <span>)
display: inline-block; - element displayed as an inline element but can apply height and width
display: grid; - element displayed as a block-level grid container

Positioning

Property Description
z-index: - sets order for elements along the z-axis
position: static; - default type of positioning
position: relative; - similar to static but can modify: top, bottom, left and right positions
position: fixed; - always stays in the same place even when the page is scrolled
position: absolute; - positioned relative to the nearest ancestor
posiion: sticky; - positioned based on scroll
float: - determines how far left/right and element should float

Colors

Property Description
color: name; - name keywords can be used to set color property values
#ff0000 / #f00 - hexadecimal value, repeated values can be shortened
hsl(200, 70%, 50%); - hue, saturation, lightness values ranged from 0 to 360
rgb(249, 2, 171); - red, green, blue values ranged from 0 to 255
hsla/rgba - fourth value ranged between 0.0 and 1.0 to determine transparency
color: transparent; - used to create fully transparent elements

Typography

Property Description
font-style: - determines how text will appear, can be used to make text italic
line-height: - desclares the vertical spacing between lines of text
@font-family {} - allows external fonts or font files to be imported directly

Git

Filesystem

Tag Name Description
$ pwd Print Working Dir. - displays the file path from the root directory
$ ls List - lists the contents of a directory
$ mkdir Make Directory - creates new directory in the filesystem
$ cd Change Directory - move throughout the filesystem. ../.. to move back up
$ touch New File - used to create new files

Git Commands

Tag Name Description
$ git init Init - creates a new repository
$ git add . Add - moves modified file to a staging area
$ git commit -m Commit - submits changes to the local repository
$ git push Push - uploads local repository to a remote one
